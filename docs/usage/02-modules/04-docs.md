# Docs — 文档

Docs 是项目的富文本文档系统，类似 Notion，支持多标签页和 Google Drive 双向同步。

## 创建文档

在 Docs 列表页点击右上角 **+** 新建文档，输入标题。

文档支持用文件夹组织（在文档菜单里设置 folder 字段）。

## 编辑器功能

### 基础格式
工具栏提供常用格式：**加粗**、*斜体*、下划线、Link、表格、图片宽度调整。

### 标注颜色
选中文字后点击工具栏的 A 按钮（红/黄/灰），给文字加底色标注。

### 引用论文（Wiki Link）
在编辑器里输入 `[[` 触发论文搜索弹窗，选择后插入 `[[citationkey]]`。

- 悬停显示论文标题/作者/年份卡片
- 点击打开论文详情

### 引用文档
输入 `{{` 触发文档搜索，插入文档内链。

### 粘贴截图
直接 `Ctrl+V` 粘贴截图，自动上传并插入图片。支持调整显示宽度（S/M/L/Full）。

### Callout 块
工具栏点击 **💡 Callout** 插入高亮块：

:::note
这是一个 Note callout — 适合用于补充说明。
:::

:::warning
这是一个 Warning — 适合用于注意事项。
:::

点击 emoji 可切换类型（Note / Tip / Warning / Danger）。鼠标悬停右上角出现 **×** 按钮整块删除。

### 数学公式
工具栏 **$x$** 插入行内公式，**$$x$$** 插入块级公式（LaTeX 语法）。

## 多标签页

一个文档可以有多个标签页（如"背景"、"方法"、"结论"）：
- 点击右上角 **+** 添加标签页
- 双击标签页标题重命名

## Google Drive 同步

文档详情页右上角有 Drive 同步按钮：

| 操作 | 说明 |
|---|---|
| **linked / new** | 推送到已绑定的 Drive 文档，或新建并绑定 |
| **existing link** | 粘贴已有 Drive 文档 URL 绑定 |
| **Smart Sync** | 自动对比修改时间，智能推送或拉取 |

:::warning
Google Docs 不支持嵌入外部图片，粘贴的截图同步时会替换为 `[📷 alt]` 占位符。从 Drive 拉回时自动还原。
:::

## 分享文档

文档详情页点击 **Share** 按钮，生成只读分享链接（任何人无需登录即可查看）。
