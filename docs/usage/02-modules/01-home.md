# Home — 项目主页

Home 是项目的中枢，提供时间线、团队成员概览和 TODO 管理。

![项目 Home 页 — 空项目初始状态](/api/help-images/home-overview.png)

一个使用中的项目 Home 页会显示倒计时、Timeline 甘特图和 TODO 列表：

![项目 Home 页 — 有内容的完整状态](/api/help-images/home-with-content.png)

---

## 倒计时

点击页面顶部 **Click to set a deadline countdown** 设置截止日期（如 CHI 投稿日）。显示格式为天/小时/分/秒倒计时。

## 甘特图 / 时间线

可视化项目进度：

- **多轨道（Track）** — 每个研究方向或子任务一个轨道，点击 **+ Add track** 新建
- **时间条目** — 拖拽调整时间范围，双击编辑标题和颜色
- **关联文档** — 每个条目可绑定一个 Docs 文档，点击直接跳转
- **@提及** — 在时间条目里提及团队成员

## TODO 列表

点击 **+ New** 创建本周任务列表，支持：
- 关联文档（`{{` 搜索文档）和成员（`@` 提及）
- 设置截止日期
- 标记完成状态

## 成员管理

**Team access** 区域显示所有项目成员。Admin 可以：
- 点击 **Invite** 邀请新成员
- 修改成员角色（Admin / Member / Viewer）
- 移除成员

## Project Settings

（Admin 可见）管理 Google Drive 绑定和 Zotero 集成，见[项目基础设置](../01-quick-start/03-project-setup)。
