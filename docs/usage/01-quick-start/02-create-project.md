# 创建项目

登录后进入项目列表页，点击右上角 **New project**：

![项目列表 — 点击 New project](/api/help-images/project-list-new.png)

输入项目名称，点击 **Create**：

![新建项目表单 — 输入名称后点击 Create](/api/help-images/project-create-form.png)

项目创建成功，点击项目名称进入：

![项目创建完成 — 点击进入](/api/help-images/project-created.png)

---

## 项目主页概览

进入项目后你会看到 Home 页面，包含 Timeline、TODO、Team access 和 Project Settings：

![项目 Home 页 — 空项目初始状态](/api/help-images/home-overview.png)

---

## 邀请成员

在 Home 页 **Team access** 区域点击 **Invite**，输入成员邮箱。

被邀请的成员会收到一封来自 `arklabgit@163.com` 的邀请邮件：

![邀请邮件示例](/api/help-images/invitation-email.png)

:::warning
邀请邮件**很可能出现在垃圾箱**里，请提醒被邀请的成员检查垃圾邮件文件夹。
:::

如果对方尚未注册，邮件里会包含注册链接，注册后自动加入项目。
