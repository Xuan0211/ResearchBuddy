# 注册

访问 [https://research.arklab-hkustgz.com](https://research.arklab-hkustgz.com)：

![ResearchBuddy 首页](/api/help-images/landing-page.png)

点击右上角 **Get started** 或 **Log in**，进入登录页面：

![登录页面 — 点击 Register 注册新账号](/api/help-images/login-page.png)

如果你是第一次使用，点击 **Register** 注册新账号：

![注册表单 — 填写姓名、邮箱和密码](/api/help-images/register-form.png)

填写完成后点击 **Create account**，注册即完成，无需邮件验证，立即可以使用。

:::tip
注册后的邮箱同时也是 git clone 的用户名，密码也一样。
:::
