# Writing

Paper-writing workspaces (LaTeX projects).

- `Project/{id}/` — One directory per writing project
  - `main.tex` — Main LaTeX file
  - `bibs/` — BibTeX files (Zotero-synced and AI-generated)
  - `sections/` — LaTeX section files
  - `images/` — Figures and images
  - `other/` — Miscellaneous files
- `resources/` — Shared writing resources
- `utils.read_only/` — Utility scripts (system-managed)
